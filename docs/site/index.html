Capabilities

Everything runs on your hardware

No API keys, no cloud bills, no data leaving your machine.

🧠

Local LLM Reasoning

Llama 3.2 runs entirely on your CPU via llama.cpp. Bayesian context and conversation memory make it personal.

🎤

Voice Pipeline

Wake word detection → Vosk speech-to-text → LLM → Piper text-to-speech. Full voice loop, zero internet.

👁️

Screen Understanding

Moondream2 vision model sees your desktop. Ask "what's on my screen?" and get a real answer.

🖱️

Desktop Automation

AT-SPI semantic automation finds and clicks buttons by name across any GTK/Qt app. No coordinates needed.

🔧

Self-Healing

Monitors disk, memory, network, and audio. Automatically repairs crashed services and cleans up resources.

🧩

Extensible Skills

Python skill SDK with hot-reload. Write a script, drop it in, and the agent learns new abilities instantly.

Architecture

Built to be fast

Rust core for performance. Python tools for flexibility. Unix sockets for IPC.

Service Language Role
ai-distro-agent Rust Core IPC server, action routing, policy enforcement
ai-distro-voice Rust Audio capture → STT → agent → TTS
ai-distro-hud Rust Desktop overlay with agent event cards
ai-distro-vision Python Moondream2 VLM server on localhost
ai-distro-healer Python Autonomous health monitor and repair loop
ai-distro-spirit Python Telegram remote control bridge
